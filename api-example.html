Example 1: Get All Data

Retrieve all items from the civil and commercial code.

fetch('/api?code=civil_and_commercial_code')
    .then(res => res.json())
    .then(data => console.log(data));

Example 2: Filter by ID

Get a specific law item by its ID.

fetch('/api?code=civil_and_commercial_code&filter_id=1012')
    .then(res => res.json())
    .then(data => console.log(data));

Example 3: Search

Search for items containing specific text across all fields.

const searchTerm = encodeURIComponent('ห้างหุ้นส่วน');
fetch(`/api?code=civil_and_commercial_code&search=${searchTerm}`)
    .then(res => res.json())
    .then(data => console.log(data));

Example 4: Sort Descending

Get all items sorted by ID in descending order.

fetch('/api?code=civil_and_commercial_code&sort=id&order=desc')
    .then(res => res.json())
    .then(data => console.log(data));

Example 5: Pagination

Get the first 5 items with pagination.

fetch('/api?code=civil_and_commercial_code&limit=1&offset=0')
    .then(res => res.json())
    .then(data => console.log(data));

Example 6: Complex Query

Combine multiple parameters: filter by title, sort, and limit results.

const title = encodeURIComponent('สัญญา');
fetch(`/api?code=civil_and_commercial_code&filter_title=${title}&sort=id&order=asc&limit=5`)
    .then(res => res.json())
    .then(data => console.log(data));
