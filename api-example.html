Example 1: Get All Data

Fetch the complete JSON file and access all items.

‚ö†Ô∏è This fetches the entire dataset from the JSON file. The "total" count represents all items in the file. 
// Fetch complete JSON data file
fetch('api/civil_and_commercial_code.json')
    .then(res => res.json())
    .then(data => {
        const items = data.civil_and_commercial_code;
        console.log({ 
            code: 'civil_and_commercial_code', 
            total: items.length,  // Total items in the dataset

        });
    });

Example 2: Client-Side Filter by ID

Fetch the complete dataset, then filter for a specific ID using JavaScript.

‚ö†Ô∏è You must fetch the entire JSON file first, then use .filter() to find matching items. The "total" count shows how many items matched your filter.

    .then(res => res.json())
    .then(data => {
        const allItems = data.civil_and_commercial_code;
        // Filter after fetching all data
        const filtered = allItems.filter(item => 
            item.id == '1012' || item.id === '‡∏°‡∏≤‡∏ï‡∏£‡∏≤ 1012'
        );

            code: 'civil_and_commercial_code',

            data: filtered 
        });
    });

Example 3: Client-Side Search

Fetch all data, then search for items containing specific text across fields.

‚ö†Ô∏è The entire JSON file is fetched first. Search is performed client-side by checking each item's properties. 
// Fetch complete JSON, then search client-side
const searchTerm = '‡∏´‡πâ‡∏≤‡∏á‡∏´‡∏∏‡πâ‡∏ô‡∏™‡πà‡∏ß‡∏ô';
fetch('api/civil_and_commercial_code.json')
    .then(res => res.json())
    .then(data => {
        const allItems = data.civil_and_commercial_code;
        // Search after fetching all data
        const results = allItems.filter(item =>
            item.title?.toLowerCase().includes(searchTerm.toLowerCase()) ||

            searchTerm: searchTerm,
            matchingItems: results.length,  // How many matched
            totalInDataset: allItems.length,  // Total available
            data: results 
        });
    });

Example 4: Client-Side Sort Descending

Fetch all items, then sort them by ID in descending order using JavaScript.

‚ö†Ô∏è Sorting happens after fetching the complete dataset. Use JavaScript's .sort() method. 
// Fetch complete JSON, then sort client-side
fetch('api/civil_and_commercial_code.json')
    .then(res => res.json())
    .then(data => {
        const items = data.civil_and_commercial_code;
        // Sort after fetching all data (descending)
        items.sort((a, b) => 
            String(b.id).localeCompare(String(a.id))
        );
        console.log({ 
            code: 'civil_and_commercial_code', 
            total: items.length, 
            sortedBy: 'id_desc',
            data: items 
        });
    });

Example 5: Client-Side Pagination

Fetch the complete dataset, then implement pagination using .slice().

‚ö†Ô∏è All data is fetched first. Pagination is done client-side using .slice(). The "total" shows the complete dataset size, while "count" shows items in the current page. 
// Fetch complete JSON, then paginate client-side
fetch('api/civil_and_commercial_code.json')
    .then(res => res.json())
    .then(data => {
        const allItems = data.civil_and_commercial_code;
        // Paginate after fetching all data
        const limit = 5;
        const offset = 0;
        const page = allItems.slice(offset, offset + limit);
        
        console.log({ 
            code: 'civil_and_commercial_code', 
            total: allItems.length,  // Total items in dataset
            offset: offset, 
            limit: limit, 
            count: page.length,  // Items in current page
            data: page 
        });
    });

Example 6: Complex Client-Side Query

Fetch all data once, then chain multiple operations: filter by title, sort, and limit results.

‚ö†Ô∏è This demonstrates chaining multiple client-side operations. First fetch all data, then filter ‚Üí sort ‚Üí limit. The "matchingItems" shows the filtered count before limiting. 
// Fetch complete JSON, then filter + sort + limit client-side
const title = '‡∏™‡∏±‡∏ç‡∏ç‡∏≤';
fetch('api/civil_and_commercial_code.json')
    .then(res => res.json())
    .then(data => {
        const allItems = data.civil_and_commercial_code;
        
        // Step 1: Filter by title
        let filtered = allItems.filter(item =>
            item.title?.toLowerCase().includes(title.toLowerCase())
        );
        
        // Step 2: Sort by ID ascending
        filtered.sort((a, b) => 
            String(a.id).localeCompare(String(b.id))
        );
        
        // Step 3: Limit results
        const limited = filtered.slice(0, 5);
        
        console.log({ 
            code: 'civil_and_commercial_code',
            totalInDataset: allItems.length,  // Original size
            matchingItems: filtered.length,  // After filtering
            returnedItems: limited.length,  // After limiting
            data: limited 
        });
    });
